# SENTINEL

ESP32-based fire and gas safety monitor.

## Overview

SENTINEL continuously watches for fire and gas hazards using sensors wired to an ESP32, triggering an alert when unsafe conditions are detected.

## Hardware

- ESP32 microcontroller
- Flame sensor - detects open flame and infrared signature of fire
- MQ-2 gas sensor - detects smoke and combustible gases such as LPG, propane, and methane
